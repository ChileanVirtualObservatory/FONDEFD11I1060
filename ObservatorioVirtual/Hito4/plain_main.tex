\vspace{0.5cm}

\begin{abstract}
    El presente informe tiene por objeto presentar la segunda etapa de
    implementación, correspondiente a las vistas del sistema y la capa
    del controlador del observatorio virtual. Esta segunda etapa
    provee un sistema funcional, con la posibilidad de realizar las
    búsquedas en base a los protocolos \emph{Simple Cone Search}
    (SCS), \emph{Simple Image Access} (SIA) y \emph{Simple Spectral
      Access} (SSA) sobre una base de datos de prueba provistos por
    ALMA (\emph{Atacama Large Millimeter/submillimeter Array}).
\end{abstract}

\vspace{0.4cm}

	ChiVO, Simple Cone Search, SCS, Simple Image Access, SIA,
